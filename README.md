# Java Programming Practice

A collection of Java programs created while learning the fundamentals of Java programming. This repository includes examples and small console-based projects covering core Java concepts.

## 📚 Topics Covered

* Variables and Data Types
* User Input
* Operators
* Strings
* Conditional Statements
* Loops
* Methods
* Arrays
* Random Numbers
* Mini Projects (Calculator, Banking System, Number Guessing Game, Dice Roller, Temperature Converter, Shopping Cart, and more)

## 🛠️ Technologies

* Java
* JDK

## 🎯 Purpose

This repository documents my Java learning journey and serves as a reference for fundamental programming concepts.
